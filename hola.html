¿Qué es Git Hub?

GitHub es una plataforma en la nube que funciona como el mayor centro de alojamiento de repositorios Git del mundo. Permite a desarrolladores almacenar, gestionar y compartir código, facilitando el trabajo colaborativo en equipo, el control de versiones y la gestión de proyectos de software. Es considerada la red social de programadores más popular.
